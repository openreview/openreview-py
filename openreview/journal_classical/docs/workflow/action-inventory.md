# Action Inventory

This page gives reviewers one place to find visible workflow actions. Detailed
behavior belongs in the linked action pages.

| Action | Main Surface | Owner Doc | Effect |
| --- | --- | --- | --- |
| Submit paper | Venue landing and Author Console | [Lifecycle](lifecycle.md) | Creates a JMLR submission with metadata and files. |
| Create assignment pages | EIC Pending Tasks | [Assignment pages](assignment-pages.md) | Prepares paper-specific AE and reviewer assignment pages; does not assign anyone. |
| Assign Action Editor | EIC Pending Tasks and paper page | [Assignment pages](assignment-pages.md) | Opens the paper-specific AE assignment page. |
| Assign selected action editor | AE assignment page | [Assignment pages](assignment-pages.md) | Assigns a handling AE through checked assignment; changing an existing handling AE requires removing the current AE first. |
| Assign/modify reviewer assignments | AE/EIC console links and paper reviewer assignment page | [Reviewer assignment](actions/reviewer-assignment.md) | Opens the paper-specific Assign Reviewers page; reviewer additions, invitations, due dates, and removals happen there. |
| Submit review | Reviewer paper page | [Review](actions/review.md) | Records reviewer comments, recommendation, and optional review file. |
| Contact Action Editor | Author or reviewer paper page | [Contact Action Editor](actions/contact-action-editor.md) | Sends a restricted paper-specific communication record to the handling AE. |
| Decision | AE/EIC paper page | [Decision](actions/decision.md) | Records and releases the editorial outcome when the workflow permits it. |
| Reviewer Rating | Assigned AE paper page and reviewer-rating launcher; EIC read access | [Reviewer rating](actions/reviewer-rating.md) | Records optional reviewer quality and timeliness metadata. |
| Camera-ready revision | Author paper page | [Camera-ready revision](actions/camera-ready-revision.md) | Collects final files and publication metadata after acceptance. |
| Verify camera-ready revision | AE/EIC paper page | [Camera-ready revision](actions/camera-ready-revision.md) | Approves final material or asks authors for changes. |
| Download publication files | Production Editor console and eligible paper page | [Publication controls](actions/publication-controls.md) | Downloads accepted publication material for production handling when publication export is enabled. |
| Mark as published | Production Editor console and eligible paper page | [Publication](actions/publication.md) | Records publication completion when OpenReview publication is enabled. |
| Retract publication | Production Editor console and eligible paper page | [Publication controls](actions/publication-controls.md) | Opens the retraction confirmation path for published material. |
