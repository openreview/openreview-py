# Production Editor Console

The Production Editor Console owns publication handling after camera-ready
material is approved.

| Area | Shows | Expected Action |
| --- | --- | --- |
| Pending publication | Accepted papers that need production handling. | Download files, inspect publication material, and mark publication complete when ready. |
| Published | Papers already marked as published. | Inspect final publication status or use allowed publication-state controls. |

Production Editor controls should not change earlier editorial decisions,
reviews, assignments, or camera-ready verification outcomes.

Pending publication rows should show the paper identity, publication status,
download entry point, and mark-as-published action when the role and paper stage
permit them. Published rows should remain useful for inspection after
publication completion, including retracted or terminal publication states.

Production Editor tables must not expose review-stage assignment controls,
reviewer identities, decision controls, or camera-ready scientific verification
controls unless another allowed role context separately grants that access.
