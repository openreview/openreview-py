# Contact Action Editor

Contact Action Editor is a restricted paper-specific communication path to the
handling Action Editor.

| Sender | Purpose |
| --- | --- |
| Author | Ask paper-specific questions or provide clarification when the action is available. |
| Reviewer | Ask review-stage questions to the handling AE. |

The action should not expose reviewer identities to authors or other reviewers.
It should be visible only when the sender role and paper state allow it.

## Visibility And Side Effects

- Authors may use the author-to-AE contact action only while the paper state
  exposes that author communication path.
- Assigned reviewers may use Contact Action Editor only while reviewer-to-AE
  contact is active for that paper.
- Submitted contact messages are private paper-specific records for the sender
  context, the handling Action Editor, and Editors-in-Chief.
- Contact messages must not be readable by peer reviewers, unrelated reviewers,
  unrelated authors, Production Editors, or the public.
- The action sends the intended handling-AE notification when submitted.
